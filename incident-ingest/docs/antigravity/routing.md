# Routing Policy

- research → Research / Source-of-truth (Gemini + browsing)
- design → Architect (Gemini)
- backend → Backend Builder (Codex)
- frontend → Frontend Builder (Codex/Gemini)
- review/security/perf → QA Reviewer (Codex)
- release/deploy → Integrator (Gemini)

## Auto-review rule
Every ticket that changes code must create a paired `review` ticket.
